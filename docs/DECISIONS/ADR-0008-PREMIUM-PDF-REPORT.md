# ADR-0008: Premium PDF report

- Status: Accepted
- Phase: P5
- Report profile: `celestial-premium-report-p5-v1`

## Decision

Celestial ASTRO AI generates a premium PDF only after recalculating the chart on
the server from the submitted birth details. The PDF renderer never trusts
planetary placements, interpretations, or receipts supplied by the browser.

The report uses the approved Celestial Observatory visual system:

- midnight navy background
- warm ivory text
- antique-gold hierarchy
- restrained indigo and cyan accents
- serif display typography with a clean sans-serif data layer
- chart-first composition and visible receipt identity

## Report contract

Timed charts include:

1. premium cover and trust contract
2. chart overview and celestial fingerprint
3. planetary positions and current Mahadasha context
4. core, career, and relationship interpretation packs
5. evidence used, rule identifiers, confidence, and limitations
6. complete calculation receipt
7. honest scope and safety limitations

Unknown-time charts receive a different report path. They show date-wide
planetary ranges, stable placements, and explicit suppression of Ascendant,
houses, exact Dasha dates, and time-dependent interpretations.

## Trust and privacy

- The generated report must carry the same `chartId` as its interpretation
  evidence.
- Only rules approved by the P4 evidence contract may appear.
- Reports are generated on demand and are not stored by the report endpoint.
- The response is private, non-cacheable, and download-only.
- The report does not claim scientific validation of astrology or guaranteed
  predictions.
- Medical, legal, financial, and mental-health certainty is prohibited.

## Verification

Automated tests load the generated bytes as a real PDF and verify:

- a valid PDF header
- minimum page coverage for timed and unknown-time reports
- report metadata
- receipt-linked report subject
- server-facing premium report profile

Visual QA renders a representative report to page images and checks typography,
page transitions, table alignment, legibility, and the absence of clipping.
